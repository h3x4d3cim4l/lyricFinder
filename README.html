Instalacja

Python

Python jest dostępny do pobrania za darmo tutaj (zalecane wersje 3.10 >=)

Pozostałe zależności

>cd ./proj
>python -m pip install -r requirements.txt --user

Uruchomienie

>cd ./proj (jezeli uzytkownik nie znajduje sie w podanym folderze)
>python main.py

po wykonaniu powyzszych poleceń aplikacja powinna być dostępna pod 127.0.0.1:5000 (w przypadku zajęcia portu Flask domyslnie wybierze inny, link bedzie widoczny w oknie wiersza polecenia)
